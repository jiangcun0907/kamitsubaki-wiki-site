## 概述

春猿火是 KAMITSUBAKI STUDIO 旗下的虚拟歌手，也是 V.W.P 成员之一。她的个人风格在神椿成员中相当鲜明，常以高能量、高张力与带有冲撞感的歌唱表现被辨认出来。

相比偏冷感或偏叙事抽离的路线，春猿火更容易给人“情绪先冲出来”的印象。她既能用近乎台风般的爆发力推动歌曲，也能在更稚气、更直接的段落里维持强烈的临场感，这种变幻自在的表现力正是她个人形象的重要组成部分。

## 角色与创作定位

官方介绍常把春猿火描述为跨越类型边界的虚拟歌手。她的核心不在于固定曲风，而在于声音里自带的情绪速度与冲击力。无论是偏摇滚、偏电子，还是更接近角色歌曲语境的作品，她都倾向于把“唱”的动作本身推到前景。

在 V.W.P 内部，春猿火往往承担更具爆发性的能量位置，这也让她与其他成员的派生曲、对唱曲常具有很强的对比张力。

## 基本资料与人物设定

春猿火早期以 Virtual Rapper 身份被介绍，公开人设由穂竹藤丸设计。她的视觉识别点是黑色长发中的红蓝挑染、金色眼睛、耳机、眼影以及机能风与街头感并存的服装。萌百资料中出现的“多重人格”等萌点属于角色和粉丝文化的表现方式，不能当作现实人格诊断。

| 项目 | 内容 |
| --- | --- |
| 角色设计 | 穂竹藤丸 |
| 主要合作 | たかやん、V.W.P 成员、穂竹藤丸等 |
| 早期原创曲 | 《逆転》《Lift Up》《猛進》《オオゴト》《告げ口》《台風の眼》 |
| 代表专辑 | 《心眼》 |
| 声音延伸 | 音乐同位体「羽累 / HARU」 |

「春猿火自由律」是她早期很有辨识度的短篇企划：把日常感受和即时想法交给说唱形式表达。它与正式发行的原创曲不同，但对于理解春猿火如何把个人语言转成音乐很重要。

## 活动历程

- **11月15日**：上传首个短视频，正式开始活动。
- **12月25日**：发布首支原创单曲《逆転》。

### 2020年

- **3月23日**：在花谱 1st LIVE「不可解（再）」中以 3D 模型首次亮相。
- **7月30日**：发布第四支原创单曲《オオゴト》。
- **9月28日**：发布第五支原创单曲《告げ口》。
- **12月5日**：举办首次线上翻唱演唱会「シュークリームライブ」。

### 2021年

- **8月27日**：于 HULIC HALL TOKYO 举办首场个人演唱会「シャーマニズム」。
- **10月6日**：首张个人专辑《心眼》正式发售。

### 2022年

- **1月28日**：发布原创单曲《Oarana》（动画《地球外少年少女》主题曲）。
- **8月6日**：举办线上翻唱演唱会「シュークリームライブ2」。
- **11月6日**：与幸祜、CIEL 共同举办「Singularity Live 2」，同场发布新曲《自由までの距離》并公开音乐同位体「羽累」的 3D 模型。

### 2023年

- **11月4日**：举办 SINKA LIVE SERIES EP.Ⅲ 春猿火 2nd ONE-MAN LIVE「シャーマニズムⅡ – 神椿市弐番街 -」。

### 2024年

- **4月10日**：作为「神椿新宿战线」的一环，举办 3rd ONE-MAN LIVE「シャーマニズムⅢ」。
- **8月**：参演虚拟舞台剧《御伽噺(染)》（以「ver0.92_prototype」首次亮相，后续参演各升级版本）。

### 2025年

- **8月27日**：发布原创单曲《(A)letheia》（免费恐怖 ADV 游戏《春琉ト怪夜》主题曲）。
- **11月2日**：出演「KAMITSUBAKI FES '25」。

### 2026年

- **6月24日**：Remix 专辑《GLITCH STAGE》正式发售。
- **9月5日 - 6日**：预定出演「KAMITSUBAKI WARS 2026 神椿横浜戦線」。

## 音乐作品与现场活动

春猿火的作品以强烈的节奏推进、说唱与摇滚式爆发为主要识别点。早期《逆転》《猛進》《オオゴト》《告げ口》建立了她的 Virtual Rapper 形象；《居場所》《覚醒 feat. さなり》《哀愁さえも仲間》则把个人情绪、合作演唱与更宽的流行编曲纳入同一条路线。

| 作品/系列 | 内容 | 说明 |
| --- | --- | --- |
| 《心眼》 | 2021 年首张个人专辑 | 汇总早期原创曲与高能量演唱方向 |
| 「シャーマニズム」系列 | 个人 one-man live | 以现场冲击力和强节奏编排为核心 |
| 「シュークリームライブ」系列 | 翻唱演唱会 | 展示她在原创曲之外的音域与合作能力 |
| 「春猿火自由律」 | 短篇说唱企划 | 将日常感受、即兴语气与说唱表达结合 |
| 「GLITCH STAGE」「SPICE OF LIFE」 | Remix/Mix 企划 | 观察原曲在不同制作语法中的重组方式 |

她与幸祜的《ATELIER》《Nectar》、与ヰ世界情緒的《LOVEぃ》《BREATHE》以及 V.W.P 派生曲，能够清楚呈现她在组合中的“推进者”位置：并非只负责高音或强音，而是把歌曲的情绪速度直接推向前景。

## 形态、设定与轶事

春猿火的视觉设计由穂竹藤丸负责，黑发中的红蓝挑染、金色瞳孔、耳机与机能风服装共同构成了她的识别系统。她在花譜「不可解・再」中首次以 3D 模型亮相，之后又在个人演唱会和大型神椿活动中逐步更新舞台形态。

“Virtual Rapper”是她早期活动的重要定位，但不等于作品全部属于传统说唱；她也会在抒情段落、合唱和动画主题曲中使用更宽的歌唱方式。音乐同位体「羽累 / HARU」则是基于她歌声数据的合成声库，应与春猿火本人分开记录。

## 现场活动与相关人物

「シュークリームライブ」及其后续场次是春猿火翻唱与合作表达的重要系列；「シャーマニズム」则是个人 one-man live 的核心品牌。她还参与「Singularity Live」和 V.W.P 大型现场，在双人或团体编排中往往承担把歌曲推向高潮的声部。

たかやん参与了不少早期说唱歌词与「春猿火自由律」内容，穂竹藤丸负责角色设计和相关视觉表达。萌百资料中常把春猿火描述为“台风般强力、又像天真少女一样变化无常”，这属于对歌唱表现的概括，不应收窄成固定曲风标签。

## 相关企划 / 关联设定

春猿火与 V.W.P、神椿大型 live 系列，以及音乐的同位体“羽累 / HARU”都有关联。对许多观测者来说，她的重要性不只是“另一个成员”，而是神椿声音版图里偏向冲击、爆燃与速度感的那个极点。
