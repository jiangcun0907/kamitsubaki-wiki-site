## 概述

ヰ世界情緒是 KAMITSUBAKI STUDIO 旗下的虚拟歌手、创作者，也是 V.W.P 成员之一。她的个人定位在神椿成员中十分独特：不仅以歌手身份活动，也持续把插画、旁白、声优式表演与更浓厚的个人创作意识带入自己的表达体系。

如果说花譜更像神椿“观测”系统的起点，理芽偏向现代流行的切口，那么ヰ世界情緒的魅力往往来自她所营造的“另一个世界”本身。她的名字、声线、视觉与叙事语感，几乎总是一起工作。

## 角色与创作定位

官方页面将她定位为“虚拟暗色系歌手”，这一说法并不只是风格标签，更接近一种完整的创作方向。她的演唱常在脆弱与强韧之间摇摆，歌曲里的黑暗、神秘与叙事密度，往往比单纯的曲风分类更重要。

此外，ヰ世界情緒并不把自己限制在“只负责唱歌”的角色里。插画、配音、讲述和世界观式表达，都是她个人身份的一部分，这使她在神椿内部具有很强的创作者属性，被视为独一无二的综合型表达者。

## 基本资料与人物设定

ヰ世界情緒的角色名保留了“ヰ”这一旧字形，是其视觉和文字识别的一部分。她公开资料中的核心设定包括象牙白发、左右不同颜色的瞳孔、遮单眼发、风衣、手套和花朵意象；不同形态会在发色、衣装和花瓣结构上变化。

| 项目 | 内容 |
| --- | --- |
| 角色设计 | 初期由 orie 负责，后续形态与演出设计涉及れおえん等创作者 |
| 创作身份 | 歌手、插画创作者、角色表达者、叙事型虚拟艺术家 |
| 代表形态 | 普遍体、Anemone 系列、花魁鸟、八咫乌、Heliotrope 等 |
| 重要声源关联 | 音乐同位体「星界 / SEKAI」 |
| 个人兴趣 | 纯音乐、电影/游戏原声、Vocaloid 与绘画创作 |

普遍体是她日常账号、联动立绘和多数曲绘中最常用的形态；现场专用形态则与具体 Live 的叙事和舞台主题绑定，不能把所有形态都当成同时存在的日常服装。

## 活动历程

### 2019年

- **12月9日**：初次上传视频，预告正式开始活动。
- **12月12日**：发布第一首翻唱曲《夜顔の告白》。

### 2020年

- **2月23日**：发布第一首原创单曲《物語りのワルツ》（Samayuzame词曲）。
- **4月27日**：发布第二首原创单曲《ジオラマドラマ》（Samayuzame词曲）。
- **7月16日**：发布第三首原创单曲《ハイドレンジア》（Toucasa词曲）。
- **9月26日**：发布第五首原创单曲《とめどなき白情》（Harumakigohan词曲）。
- **12月26日**：举办首次 Cover Live「キャンディライブ」。会后发布第七首原创单曲《シリウスの心臓》（伞村Tota词曲），该曲成为其最具代表性的单曲之一。

### 2021年

- **3月13日**：作为虚拟歌手组合 V.W.P（Virtual Witch Phenomenon）的成员，于花谱 2nd ONE-MAN LIVE「不可解弐Q2」正式公开亮相并宣布组合结成。
- **10月23日**：举办首次个人 3D 虚拟单独演唱会 / 1st ONE-MAN LIVE「Anima」。同日宣布制作首张个人专辑《創生》及音乐同位体「CeVIO AI 星界」。
- **12月8日**：发行首张个人专辑《創生》。

### 2022年

- **4月29日**：以其歌声数据为基础开发的音乐同位体「CeVIO AI 星界 / SEKAI」正式发售。
- **7月17日**：与理芽联合举办虚拟演唱会「Singularity Live Vol. 1」，并公开合作单曲《パンドラコール》（Pandora Call）。
- **9月10日 - 10月10日**：在东京举办首次个人艺术展「ヰ世界情緒展」。

### 2023年

- **1月14日**：举办第二次 Cover Live「キャンディライブ2」。
- **6月18日**：举办第二次个人 3D 虚拟单独演唱会 / 2nd ONE-MAN LIVE「Anima Ⅱ -神椿市参番街-」，并公布全新 3D 造型「普遍体」。
- **12月9日**：在 4 周年直播中宣布制作第二张个人专辑《色彩》。

### 2024年

- **3月27日**：第二张个人专辑《色彩》正式发售。
- **8月7日**：于横滨 Pacifico 国立大厅举办首次线下个人演唱会 / 3rd ONE-MAN LIVE「Anima Ⅲ」。
- **8月13日 - 15日**：作为 V.W.P 成员出演 KAMITSUBAKI STUDIO 首部成长型虚拟舞台剧《御伽噺（染）》的首次公演。

### 2025年

- **2月 - 3月**：在东京与大阪举办第二次个人艺术展「ヰ世界情緒展Ⅱ」。
- **9月25日 - 10月26日**：首次在海外举办个人展，于上海举办「ヰ世界情緒展」特展。

### 2026年

- **2月25日**：发行首张个人 EP《幻视录》。
- **3月19日**：自制并以其形象“水濑星海”为主角的虚拟偶像养成冒险游戏《Virtual Ties ～ヰ世界情緒夢想曲～》在 Steam 正式发售。
- **3月25日**：举行 YouTube 频道 50 万订阅纪念直播。
- **5月1日 - 2日**：在 Zepp Haneda 举办首个 2DAYS LIVE（DAY-1「Flower Closet」 / DAY-2「Anima Re:birth」）。

## 音乐作品与现场活动

ヰ世界情緒的个人作品具有明显的“歌曲—绘画—角色—世界观”联动特征。早期原创曲《物語りのワルツ》《ジオラマドラマ》《ハイドレンジア》《とめどなき白情》逐步建立叙事密度；《シリウスの心臓》则成为个人音乐的重要入口。

| 作品/系列 | 内容 | 说明 |
| --- | --- | --- |
| 《創生》 | 2021 年首张个人专辑 | 将歌唱、绘画和“创世”意象集中到一张作品中 |
| 《色彩》 | 2024 年第二张个人专辑 | 延续个人世界观并扩大创作者合作范围 |
| 《幻视录》 | 2026 年个人 EP | 体现后续阶段更凝缩的声音与视觉表达 |
| 「Anima」系列 | 个人 3D/线下演出 | 将角色形态、舞台与叙事推进结合 |
| 「Flower Closet」「Anima Re:birth」 | 2026 年 2DAYS LIVE | 对个人演出历史进行并置和再诠释 |

她还参与《Virtual Ties ～ヰ世界情緒夢想曲～》等跨媒体项目，在其中承担角色原型、视觉创作者和歌手等多重身份。与理芽共同举办的 Singularity Live、V.W.P 的团体曲，以及以星界为代表的同位体项目，则分别对应合作演出、团体叙事和声音合成三个方向。

## 形态、设定与轶事

ヰ世界情緒的角色设计与形态演进涉及多位创作者。普遍体是日常活动和多数曲绘中最常见的形态；Anima 系列及其他现场则会使用根据歌曲和舞台概念变化的形态。她本人也持续参与曲绘、角色表达和视觉创作，不能只以“演唱者”概括。

公开内容中，她经常谈到对纯音乐、电影与游戏原声、Vocaloid 作品以及不同年代流行音乐的兴趣；绘画也是其创作实践的重要部分。关于饮食、宠物和个人生活的趣闻应作为公开访谈或节目语境中的轶事阅读，不应与正式人物设定混写。

## 现场活动与相关人物

ヰ世界情緒的 Live 体系包括「キャンディライブ」及其后续 Cover Live、「Anima」系列个人演出、「parallel canvas」Mini-Live，以及与理芽的「Singularity Live」。这些演出分别突出翻唱、3D 形态、绘画和角色叙事，构成她区别于单纯音乐频道的跨媒体履历。

她的视觉创作和曲绘参与度很高，作品资料中经常同时出现 orie、れおえん、samayuzame、とうかさ、はるまきごはん等名字。关于她喜欢意面、绘画、养鱼或游戏的内容属于节目和访谈轶事，可以作为人物阅读材料，但不应与官方基本资料混为一谈。

## 相关企划 / 关联设定

ヰ世界情緒与 V.W.P、音乐的同位体“星界 / SEKAI”、以及神椿市相关世界观企划都有紧密连接。她在神椿内部的重要性，常常不止于“成员之一”，而更像是把歌手、角色与创作者三种身份叠在一起的复合型节点。
